# Manpower Management System

A modern frontend-only manpower management platform built using React + Vite for handling staff operations, vendor management, center tracking, salary management, and CBT exam workforce coordination.

---

# 🚀 Features

## 👨‍💼 Staff Management
- Add/Edit/Delete Staff
- Staff Search
- Aadhaar Number Support
- DOB & Location Tracking
- Responsive Staff Cards

## 🏢 Vendor Management
- Vendor Tracking
- Vendor Pending Calculation
- Payment Status Management

## 🏫 Center Management
- Center Information Management
- Total System Count
- Booked System Count
- Center Image Upload
- Local Image Storage

## 📋 Duty Assignment
- Staff Duty Allocation
- Shift Management
- Assignment Updates

## 💰 Salary Management
- Full Payment Support
- Partial Payment Support
- Pending Salary Tracking
- Payment History

## 📊 Dashboard Analytics
- Salary Overview
- Vendor Pending Overview
- Staff Summary
- Financial Tracking

## 📁 Reports & Backup
- CSV Export Reports
- JSON Backup
- JSON Restore
- Offline Data Support

---

# 🛠 Tech Stack

- React.js
- Vite
- JavaScript
- CSS
- localStorage

---

# ⚡ Project Highlights

- Fully Frontend-Only
- No Backend
- No API Integration
- Offline Usable
- Responsive Design
- Lightweight & Fast
- LocalStorage Data Persistence
